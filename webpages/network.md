# Getting Started with Network Automation

The Ansible Network Automation Workshop is a 90 minute, hands-on workshop covering Ansible Automation with respect to routers and switches. Attendees will learn how to pull facts from devices, build templated network configurations, and apply these concepts at scale with Ansible Automation Platform. Youâ€™ll put it all together by exploring the job templates, surveys, access controls and more.

Note: In this workshop, attendees will be managing network devices, specifically a Cisco IOS-XE router.

> **IMPORTANT TO NOTE** 
> 
> This workshop is also known as the Network Automation 90-min Technical Workshop
